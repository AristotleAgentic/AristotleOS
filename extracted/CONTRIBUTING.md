# Contributing to AristotleOS

This workspace follows the repository root contribution policy. AristotleOS is
open for public review, security diligence, documentation improvements,
framework adapters, governance patterns, and implementation work.

Every commit must include a Developer Certificate of Origin sign-off:

```text
Signed-off-by: Your Name <you@example.com>
```

Unless you and AristotleAgentic have a separate written agreement, your
contribution is submitted under the same license that applies to the file,
package, or directory you are changing.

- MPL-2.0 source files receive MPL-2.0 contributions.
- Apache-2.0 packages receive Apache-2.0 contributions.
- CC-BY-4.0 documentation receives CC-BY-4.0 contributions.

Reviewers are especially encouraged to examine the Commit Gate, Warrant binding,
Governance Evidence Ledger replay, mesh/disconnection story, and adapter refusal
before emission.

See the root `CONTRIBUTING.md` for the full Developer Certificate of Origin text.
