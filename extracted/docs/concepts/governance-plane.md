# Governance Plane

The Governance Plane is the runtime control layer between autonomous intent and execution.

It exists to prevent agents, robots, workflows, or infrastructure automation from holding standing power over consequential systems. It resolves authority, compiles policy, evaluates the Commit Gate, issues warrants, and commits evidence.

Developers use it by sending action intent to AristotleOS before invoking the underlying tool or system.

It produces decision records, warrants when allowed, and GEL evidence for replay.
