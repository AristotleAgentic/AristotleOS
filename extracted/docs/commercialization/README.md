## Verification

Run:

```bash
npm run verify:uav-swarm-proof
```

The verifier prints a plain-English pass/fail report for the proof sequence: 40 assets, partition, degraded authority, allowed fallback, refused mission expansion, expired/stale/revoked/review-required classifications, reconnect, reconciliation, and GEL/evidence entries.

## Scope

This package describes a simulated governance proof demo. It does not claim live aircraft control, FAA certification, airworthiness certification, weapons-system authorization, or replacement of certified autonomy/control stacks.
